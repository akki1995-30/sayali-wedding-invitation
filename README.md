# 💒 Sayali & Akash Wedding Invitation

A beautiful, animated wedding invitation website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## ✨ Features

- 🎨 Elegant pink/rose theme matching traditional Indian wedding aesthetics
- 🌸 Animated floating petals and sparkles
- 💍 Smooth entrance animations using Framer Motion
- 📱 Fully responsive design (mobile-first)
- 🕉️ Hindi text support with proper fonts
- 🎭 Beautiful typography with Google Fonts

## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Fonts:** Google Fonts (Great Vibes, Playfair Display, Noto Sans Devanagari)

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

```
src/
├── app/
│   ├── globals.css      # Global styles and animations
│   ├── layout.tsx       # Root layout with metadata
│   └── page.tsx         # Main invitation page
├── components/
│   ├── Header.tsx       # Ganesh blessing & wedding title
│   ├── CoupleSection.tsx # Bride & Groom cards
│   ├── DateSection.tsx  # Wedding date & time
│   ├── VenueSection.tsx # Venue & invitation message
│   ├── Footer.tsx       # Decorative footer
│   ├── FloatingPetals.tsx # Animated petals background
│   └── Sparkles.tsx     # Animated sparkles
```

## 🎨 Customization

To customize the invitation:

1. **Names & Details:** Edit the text in respective component files
2. **Colors:** Modify the color palette in `tailwind.config.js`
3. **Animations:** Adjust timing and effects in `globals.css` and components

## 📦 Build for Production

```bash
npm run build
npm start
```
